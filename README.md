# QueenBot

بوت ذكي متعدد المهام يتم تطويره بخطوات منظمة لضمان الأداء والاستقرار.  
هذا المستودع يحتوي على الملفات الأساسية الخاصة بالمشروع وسيتم تحديثه تدريجياً.

## الميزات القادمة
- نظام قوائم احترافي
- تكامل مع Termux
- دعم التحديثات الديناميكية
- نظام اقتصادي متطور
